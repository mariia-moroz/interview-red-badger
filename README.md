# Martian Robots

## Requirements

- Node.js
- npm

## Install

```bash
npm install
```

## Run

The app reads input from a file.

Use the default sample file:

```bash
node index.js
```

Use a specific input file:

```bash
node index.js sample-input.txt
```

Content of input file example:
```bash
5 3
1 1 E
RFRFRFRF
3 2 N
FRRFLLFFRRFLL
0 3 W
LLFFFLFLFL
```

## Test

```bash
npm test
```
